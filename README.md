# My Pantry: Solusi Cerdas Manajemen Persediaan Bahan Makanan untuk Rumah Makan

My Pantry adalah aplikasi manajemen persediaan bahan makanan yang dirancang khusus untuk membantu rumah makan meningkatkan efisiensi operasional secara akurat. Dengan My Pantry, dapat:
1. Memantau stok secara real-time
2. Mengelola berbagai jenis bahan makanan
3. Dapatkan notifikasi bahan kadaluwarsa
4. Pantau riwayat transaksi

## Fitur Utama:

1. **Katalog Bahan Makanan**:
   Pihak rumah makan dapat membuat daftar lengkap bahan makanan yang digunakan di rumah makan.

2. **Kelola Persediaan**:
   Memungkinkan pihak rumah makan untuk menambah, mengedit, atau menghapus item dari persediaan dengan mudah.

3. **Notifikasi Kadaluwarsa**:
   Mendapatkan peringatan otomatis saat bahan makanan hampir atau sudah kadaluwarsa.

4. **Riwayat Aktivitas**:
   Melacak semua aktivitas terkait persediaan bahan makanan dan dapat dicetak.

## Teknologi yang Digunakan:

- **Flutter**: Untuk pengembangan aplikasi lintas platform (Android dan iOS).
- **GetX**: Sebagai state management dan navigasi.
- **API Integration**: Menggunakan Dio untuk komunikasi dengan backend.
- **Penyimpanan Lokal**: Memanfaatkan GetStorage untuk menyimpan data offline.

## Keunggulan:

1. **Antarmuka Pengguna Ramah**:
   Dirancang dengan mempertimbangkan kebutuhan pihak rumah makan.

2. **Performa Tinggi**:
   Menggunakan teknologi terkini untuk memastikan aplikasi berjalan lancar.

3. **Keamanan Data**:
   Menerapkan praktik keamanan terbaik untuk melindungi informasi pengguna.
